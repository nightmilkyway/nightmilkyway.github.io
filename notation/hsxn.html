Hyperstar X-Notation

Overcomplexed and very powerful notation with an unimaginable growth speed.

Current Extensions

  • X-Notation
  • Extended X-Notation
  • Superextended X-Notation
  • Hyperextended X-Notation

TAD

  • Iterator - a set of elements or other iterators (denoted as [...])
  • Element - positive natural number
  • Base - first element
  • Recursion - repeating of an array 'base' times.
  • Iteration - repeating of elements of array 'base' times.
  • # - a part of a set that does not transformed.
  • $ -repeating of element
  • γ - base.

All rules from previous array types is applicable to arrays of follow types, which contain iterators of previous type.

Superscript for "#" mean length of part of a set.

Superscript array for figure braces equal to the recursion of array.

Superscript array for square braces equal to the recursion of iterator (like this [[[...]]]).

Analysis of this notation you can see here.

X-Notation

Basic notation

  • X { n } = n + 1
  • X { # ' 1 } = X { # }
  • X { n } x = X { n+1 } x-1
  • X { # ' { n } } = X { # ' n + 1 }
  • X { # x ' 1 ' n ' # } = X { # x ' γ ' n - 1 ' # } $ x 0 , γ
  • X { n [ 1 ] } = X { $ n n }
  • X { n [ x ] } = X { $ n n [ x-1 ] }

Extended X-Notation

Iterator Notation (Iterators themselves can have sets, and they have the same rules as regular sets.)

  • X { n [ 1 ] ' 1 } = X { n [ X { [ ⋯ ] } ] }
  • X { n [ 1 ] ' [ 1 ] } = X { n [ 1 ] ' $ n n }
  • X { n [ [ 1 ] ] } = X { n $ n [ n ] }
  • X { n [ 1 ] [ [ 1 ] ] } = X { $ n n [ [ 1 ] ] }

Superextended X-Notation

Full iterators notation - iterator contain a sets with iterators.

  • X { n [ 1 \ 1 ] } = X { n [ 1 ] n }
  • X { n [ 1 \ 1 \ ⋯ ] } = X { n [ [ ⋯ ] \ ⋯ ] }

Hyperextended X-Notation

Extended iterators notation - backslash now can be indexed.

  • X { n [ 1 \ 2 1 ] } = X { n [ ⋯ ] \ [ ⋯ ] \ ⋯ }
  • X { n [ 1 \ s + 1 1 ] } = X { n [ ⋯ ] \ s [ ⋯ ] \ s ⋯ }
  • X { n [ 1 \ 1 ' 1 1 ] } = X { n [ ⋯ ] \ n [ ⋯ ] \ n ⋯ }
