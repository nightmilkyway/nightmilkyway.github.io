Hyperstar Hyperindex Notation

Hierarchial notation, replacement for FGH.

Hyperindex Notation

  1. I 0 ( n ) = 2 n
  2. I μ + 1 ( n ) = I μ n ( n )
  3. I μ ( n ) = I μ [ n ] ( n )

Object Expansion Rules

  1. m [ n ] = m
  2. ( @ [ 1 ] ) [ n ] = n
  3. ( # ; @ [ 1 ] ) [ n ] = ( # )
  4. ( ξ 0 + ξ 1 + ξ 2 + ξ 3 + ⋯ + ξ k ) [ n ] = ξ 0 + ξ 1 + ξ 2 + ξ 3 + ⋯ + ξ k [ n ] , only if: ξ 0 > ξ 1 > ξ 2 > ξ 3 > ⋯ > ξ k [ n ]
  5. ( ξ + 1 ; # ) [ n ] = ( ξ ; # ) + ( ξ ; # ) + ( ξ ; # ) + ⋯
  6. ( @ [ 1 ] ; 1 ; ξ + 1 ; # ) [ n ] = ( @ [ 1 ] ; < / > ; ξ ; # ) [ n ]
  7. ( @ [ 1 ] ; ξ ; # ) [ n ] = ( @ [ 1 ] ; ξ [ n ] ; # )

Expanded forms of object - any sequence of objects separated by '+' which can be obtained by using object expansion rules

  • If objects α and β are the same, α = β.
  • If object α contain object β in expanded form, α > β.
  • If object β contain object α in expanded form, α < β.
